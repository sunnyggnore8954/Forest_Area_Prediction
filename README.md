# Forest_Area_Prediction
이 프로젝트는 Python을 기반으로 개발한 지역별 산림면적 분석 및 미래 예측 프로그램입니다. Pandas와 NumPy를 활용해 CSV 형식의 지역별·연도별 산림 데이터를 불러오고 정제하였으며, 누락된 연도 데이터는 선형 보간법으로 보완하였습니다. 이후 Scikit-learn의 LinearRegression 선형 회귀 모델과 train_test_split을 사용해 과거 산림면적 데이터를 학습하고, 선택한 지역의 2021년부터 2050년까지 산림면적을 예측하도록 구현하였습니다. 예측 결과는 Matplotlib과 koreanize_matplotlib을 이용해 한글 그래프로 시각화하였으며, 미래 산림면적의 최댓값·최솟값·평균값과 해당 연도를 분석하고, 산림면적이 일정 기준 이하로 감소할 경우 탄소 배출 및 기온 변화와 관련된 경고 메시지를 제공하도록 설계하였습니다.
